HTML이란 무엇인가?

Hypertext Markup Language (HTML)is the standard markup language for creating web pages and web applications. web browsers receive Html documents form a web server or from local storage and render them into multimedia web pages. Html descrilbes the structure of a web page semantically and originally included cues for the appearance of the document.

Html elements are the building blocks of Html pages. with Html constructs, images and other objects, such as interactive forms, may be embedded into the renddered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. Html elements are delineated by tags, written using anlge brackets.
